# Lab 01 – GitHub Setup & First Push

## What this lab does
This lab introduces the basic Git and GitHub workflow required for the AI-216 course. It involves creating a structured repository, adding the first Python script, and practicing commits, branching, merging, and pushing changes to GitHub.

## Key commands used
- git init  
- git add .  
- git commit -m "message"  
- git push  
- git checkout -b week01-readme-improve  
- git merge week01-readme-improve  
- git status  

## What I learned
- How to initialize and manage a Git repository
- How to use branches to make safe changes and merge them
- How to organize a project repository professionally

## AI/ML relevance
Git is essential for AI and Machine Learning workflows because it supports reproducibility and collaboration. It helps track changes in code, experiments, and models, making it easier to work in teams and maintain consistent AI development pipelines.
